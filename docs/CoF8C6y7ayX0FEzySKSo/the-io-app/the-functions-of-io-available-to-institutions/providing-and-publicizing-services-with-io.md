# Providing and publicizing services with IO

The features that allow organizations to **provide and promote their services** on IO are:

| Feature                                | Description                                                                                                                                                                                                            | Available |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| **Services section**                   | Section containing the list of all services provided in the app, grouped by organization                                                                                                                               | ✅         |
| **Service pages**                      | Informational pages containing the description of each service, external links, and any calls to action                                                                                                                | ✅         |
| **Service templates**                  | Templates and examples to follow for providing the most common services                                                                                                                                                | ✅         |
| **New services section**               | Users will be able to apply filters and view services grouped by topic and geographic area                                                                                                                             | ✅         |
| **Featured services**                  | A dedicated area will highlight services of particular relevance or interest                                                                                                                                           | ✅         |
| **User-requested services in webview** | From the services section, users will be able to perform specific actions — such as completing a registration — on webpages provided by the organization and displayed in the IO app with a consistent user experience | ✅         |
