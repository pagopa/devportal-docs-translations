# 🎬 To get started

This manual is intended for **public bodies and public service providers** who want to integrate the **Sign with IO** service to ask citizens to sign documents and contracts via the **IO app**.

The document aims to:

- explain the **functionality** and its **benefits**;
- illustrate the **integration** methods for public bodies;
- explain how to **prepare** a document and **request its signature**;
- provide instructions on how to check the **status** of a signature and download the **signed documents**.

{% hint style="warning" %}
Only after signing the **agreement** can you test the service by generating the test **API Keys** directly from the [Reserved Area](https://selfcare.pagopa.it/auth/login).\
If the Body is already integrated with the IO App, it cannot use the API keys it already has, but must generate product-specific API Keys.
{% endhint %}
