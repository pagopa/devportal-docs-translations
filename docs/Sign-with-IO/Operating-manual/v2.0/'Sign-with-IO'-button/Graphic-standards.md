# 🎗️ Graphical standards

## How to use it?

#### Websites and Apps

You can insert the “Sign with IO” button on the page dedicated to signing documentation. The application rules are listed below.

#### Specifications

The “Sign with IO” button is composed of three main elements, which cannot be changed: logo, font and color.

#### Logo

The IO logo must always be present inside the button, without any changes in proportion or dimensional distortion.

#### Font

Next to the IO logo, the text “Sign with IO” must always be present, displayed in the "_Titillium Semibold 16 px_" font

#### Color

The background color of the “Sign with IO” button must be Blue IO 500 #0B3EE3; the elements on it, such as the logo and text, must be white.

#### Dimensions

The width of the button can be increased according to visual needs. The following dimensions remain unchanged: the height is 48px, the top and bottom padding is 12 px, and finally, the corner radius of the button is 8 px.

## How to customize the “Sign with IO” button?

#### Language

The text “Sign with IO” can be translated into the language chosen for the service.
