# Glossary

| Term              | Meaning                                                                                                                                                                                                                                                                                                                                    |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Document          | The document to be signed, including metadata (title, clauses to be signed).                                                                                                                                                                                                                            |
| Dossier           | The template for the _dossier_ that contains the set of metadata for the **Documents** to be signed within a Signature Request.                                                                                                                                                                                            |
| Signature request | <p>The signature request that is sent to a user. It is the resource that tracks the entire lifecycle of the signing process. It contains a <code>status</code>, the <code>timestamps</code> for each event, and the list of attached documents (updated according to each phase).</p><p>It is created from a <strong>Dossier</strong>.</p> |
