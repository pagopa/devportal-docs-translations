# 🕗 Changelog

### Version 2.0 (May 2026)

- Added a note on the adaptation to the eIDAS 2 regulation in the [How does it work?](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v2.0/firma-con-io/come-funziona) section
- Updated the [Graphic standards](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v2.0/pulsante-firma-con-io/standard-grafici) section
- Minor fixes

### Version 1.0 (December 2025)

- Updated the [Getting started](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0) section, removing references to the product's controlled trial
- _\[September 2023]_ Added the [Environments: Test & Production](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/api/ambienti-test-and-produzione) section, which introduces the ability to use the test and production environments simultaneously.
- _\[September 2023]_ Added the [Back Office](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/back-office/primi-passi) section, which explains how to generate and manage API Keys.
- _\[July 2023]_ Added the [Creating a Signature Request](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/richiedere-una-firma/creazione-di-una-signature-request) section, with the ability to insert specific signature fields for the signature request.
- _\[July 2023]_ Added the [Cancelling the request](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/richiedere-una-firma/cancellazione-della-richiesta-di-firma) section with the ability to cancel a signature request after sending it to the user.
- _\[May 2023]_ Added the [Sign with IO button](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/pulsante-firma-con-io/il-pulsante-firma-con-io) section.
- _\[May 2023]_ Added the [Validating documents](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/il-processo-di-firma/preparare-i-documenti/validare-i-documenti) section to verify that the document has been prepared correctly with valid signature fields.
- _\[May 2023]_ Added the [Creating the Dossier](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/creare-il-dossier) section, with the ability to enter the support contact for the specific Dossier.
- _\[February 2023]_ Added the [Checking a signature's status](https://developer.pagopa.it/it/firma-con-io/guides/manuale-operativo-fci/v1.0/verificare-lo-stato-di-una-firma) section with specifications for getting the list of created signature requests from a Dossier.
