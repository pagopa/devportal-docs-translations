# ⚒️ Environments: Test & Production

Once you have completed the service membership Agreement and logged into the [Reserved Area](https://selfcare.pagopa.it/auth/login?onSuccess=dashboard), you will always have two distinct environments available: one dedicated to **Test** and one for **Production**. This setup allows you to conduct internal tests on additional documents or contracts at no extra cost and without interfering with the operational process in production for users.

{% hint style="info" %}
Currently, you can only self-generate API Keys for the test environment. To get the API Keys for the production environment, you must complete the integration with the service.

As soon as you have completed the integration, contact us at\
**io-service-management@pagopa.it** to receive the production API Keys!
{% endhint %}
