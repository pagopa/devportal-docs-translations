# Data details: opportunities

{% hint style="info" %}
Fields marked with 👁️ will be visible in the Operator profile inside the IO app. Fields marked with 🇬🇧 must also be provided in English.
{% endhint %}

### Opportunity data

* Name; 👁️ 🇬🇧
* Description; 👁️ 🇬🇧
* Duration, including start and end date; 👁️
* Discount amount; 👁️
* Product or service category; 👁️
* Terms of use, if any; 👁️ 🇬🇧
* Link to the opportunity, if any;
* EYCA visibility, if applicable.

{% hint style="info" %}
For Operators who have chosen opportunities to be validated through the Beneficiary’s physical presence, the opportunity will be directly visible on the EYCA circuit and no confirmation will be requested.
{% endhint %}

### Data relating to recognition methods

* Static discount code, for Operators who have chosen the _static discount code method_;
* CSV file containing the list of discount codes, for Operators who have chosen the _code list method_;
* URL and `Referer` parameter to reach the page dedicated to opportunities, for Operators who have chosen the _landing page method_. 👁️
