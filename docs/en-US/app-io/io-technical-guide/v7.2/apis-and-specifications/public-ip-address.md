# Indirizzo IP pubblico

Le api di IO sono raggiungibili tramite l'host name api.io.pagopa.it che risponde all'ip 72.146.64.117

{% hint style="info" %}
Come già anticipato, a partire dal 2 luglio 2025, i servizi API della piattaforma IO sono accessibili tramite un nuovo indirizzo IP pubblico: 72.146.64.117.

Il precedente indirizzo IP (20.56.186.217) non è più attivo.
