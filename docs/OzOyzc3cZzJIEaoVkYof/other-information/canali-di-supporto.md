# Support channels

## Support for front office anomalies or PDND API issues

* Use the **“Assistance”** link in the front office header.
* You will be redirected to a **form** where you can provide a **description of the issue**, the **operational context**, and an **email address** for feedback.

## Technical reports and improvement proposals

* You can open an **issue** on the [official GitHub repository](https://github.com/pagopa/pdnd-interop-frontend/issues), preferably using the **available templates**.
* **Public issues** must **not contain confidential information** such as **client assertions**, **credentials**, or **tokens**. Any report containing sensitive data will be **removed by the administrators**. For confidential materials, please use the **assistance channels** indicated above.
