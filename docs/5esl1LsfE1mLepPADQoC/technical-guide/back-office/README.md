# Back Office

You can use PDND in two ways: via the graphical interface (_back office_) or through APIs. To access the back office, you must log in using SPID or CIE via the [Members Area](https://selfcare.pagopa.it/) platform, which streamlines user access and management for all back offices managed by PagoPA S.p.A.

The interface is organized into multiple sections:

* In the first header, you'll find links to _support_ and _documentation_.
* The second header displays, on the left, the product you’re using, and on the right, the organization you are operating for.
* On the left sidebar, there's navigation to all product pages. At the bottom, you'll find links to common Reserved Area features—like the **"**&#x55;ser&#x73;**"** menu—which allows administrators to manage users for their entity.
* The main content area (with a light gray background) displays the page content where you can perform your actions.
* The footer includes all legal notices, privacy information, accessibility details, etc.

{% hint style="info" %}
SPID and CIE are eIDAS compliant authentication methods. For more information, see [AgID's news](https://www.agid.gov.it/it/piattaforme/eidas/cie-verso-eidas) (source in Italian).
{% endhint %}
