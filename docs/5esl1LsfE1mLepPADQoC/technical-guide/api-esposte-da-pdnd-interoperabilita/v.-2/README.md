# v. 2

As already mentioned, v. 2 is the current version of the PDND core APIs, supporting both read and write operations—identical to what a user can perform through the graphical back-office interface.

To execute _write_ operations, an administrator assigned to the organization must be designated as the administrator of the client.

The v. 2 APIs are currently being rolled out. The most up-to-date API specification is always accessible online [here](https://developer.pagopa.it/pdnd-interoperabilita/api/pdnd-core-v2#/). Monthly releases are scheduled through the early months of 2026 to ensure comprehensive functionality coverage. Details of endpoints released over time are shared in the [Release notes](https://developer.pagopa.it/pdnd-interoperabilita/release-note) (Italian only).
