# Support channels

## For problems and errors encounted in the back office

A "Support" link can be found in the page header, at the top right. By clicking it, you will be redirected to a form where you can enter the details of your request and provide a contact email to receive a reply.

## For other technical issues or improvement proposals

You can open an issue in [this Github repository](https://github.com/pagopa/pdnd-interop-frontend/issues). When possible, please use the issue templates provided.

{% hint style="danger" %}
The client assertion is sensitive information. Never include it within publicly accessible Github issues. Any request containing confidential or sensitive information will be removed without notice by the administrators.
{% endhint %}
