# Service requests

Service requests are the administrative requirement that allows subscribers to register to use an e-service available in the PDND _E-service Catalog_.

Within the request, it is only verified that the consumer possesses the access requirements (called attributes) that the provider indicates as a necessary condition for subscription.

For this reason, it is possible to have only one active service request for each e-service.

As an administrative requirement authorizing the party to extract data from a provider’s e-service, this request must be completed and submitted by a user with the administrator role.
