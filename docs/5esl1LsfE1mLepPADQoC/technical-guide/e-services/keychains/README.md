# Keychains

Allows producers to sign the responses sent to consumers. Consumers can verify the integrity of the response, confirming that the signature was applied by the producer.

A keychain is a container that can be associated with one or more e-services. It is intended to hold a list of public keys, whose corresponding private keys will be used to sign responses.

The keys are uploaded by technical users known as security operators, who manage the e-service.

For more information on how to ensure response integrity for the consumer, see the [dedicated section](../../utilizzare-i-voucher/ensuring-response-integrity.md).
