# 3️⃣ Generating and managing Manage Group API keys

By accessing the Reserved Area and subsequently the IO product's Back-office, an administrator can generate an API Key pair for a group, selecting it from existing groups for which a key pair has not yet been generated.

Subsequently, the administrator can modify a Group's API Key:

- can rotate the keys: key rotation is a fundamental security practice that involves periodically renewing access credentials; this practice is not just a recommendation but often a compliance requirement;
- can modify the list of IP addresses authorized to use them;
- can delete the key pair for the group.
