# 1️⃣ Access to my groups' services

By accessing the Private Area and then the IO product's Back-office, an **operator** belonging to a group will only see (and be able to operate on) the services associated with that group.

{% hint style="info" %}
When creating a new service, an operator belonging to at least one group is required to select a group to associate the service with. Any subsequent change to this association can only be made by an administrator.
{% endhint %}
