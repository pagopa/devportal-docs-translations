# Service Topics: Get all service topics

## Description

This API allows you to retrieve the list of possible topics useful for thematically classifying a service.

{% hint style="info" %}
You must use the new [`manage`](../../Features/Publishing-a-service/Manage-key/Manage-key.md) key for service management
{% endhint %}
