# Test with real Fiscal Codes

In order to send messages to real Fiscal Codes belonging to the users themselves, you must:

<details>

<summary>If you are using the Developer Portal</summary>

- Write to [areariservata@assistenza.pagopa.it](mailto:areariservata@assistenza.pagopa.it), specifying:

  - email of the user registered for the developer portal
  - service (or subscription) id
  - the Fiscal Codes to enable for receiving push notifications and messages within the IO App

  The activation process involves sending an in-app **message containing a unique verification code** to each of the individuals identified by the provided Fiscal Codes. To complete the activation, each recipient must send back the verification code received as instructed in the message itself.

</details>

<details>

<summary>If you are using the Reserved Area</summary>

No specific action is required; your services are already enabled to send messages to the general user base.

</details>
