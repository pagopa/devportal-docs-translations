# Test sending pagoPA notices

To be able to send pagoPA payment notices, during the test phase you can write to the email address [areariservata@assistenza.pagopa.it](mailto:areariservata@assistenza.pagopa.it) specifying:

<details>

<summary>If you are using the Developer Portal</summary>

- email of the user registered on the developer portal
- service ID (or subscription)

</details>

<details>

<summary>If you are using the Back Office</summary>

- Tax code of the user registered in the Back Office
- service ID (or subscription)

</details>

{% hint style="info" %}
For the test phase, payments with a maximum amount of €0.10 will be enabled.
{% endhint %}
