# 📘 Glossary

| Term              | Definition                                                                                                                                                                                                                                                                                                                              |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Reserved Area** | Platform where entities can sign up for and use the products of PagoPA S.p.A.                                                                                                                                                                                                           |
| **Basic plan**    | Standard subscription agreement, required to use a PagoPA S.p.A. product.                                                                                                                                                                                               |
| **Premium plan**  | Subscription agreement that includes the use of "premium" features not included in the basic plan.                                                                                                                                                                                                                      |
| **Citizen**       | User of the IO app                                                                                                                                                                                                                                                                                                                      |
| **Entity**        | Public or private organization that uses PagoPA S.p.A. products to provide services to citizens.                                                                                                                                                                        |
| **IO, IO app**    | The free Italian mobile application that aims to make public administration services accessible to citizens                                                                                                                                                                                                                             |
| **Message**       | The way entities send communications to citizens via the IO app                                                                                                                                                                                                                                                                         |
| **PagoPA**        | The highly specialized public company with the mission of disseminating digital public services through the development of innovative projects and the management of strategic technological infrastructures                                                                                                                            |
| **pagoPA**        | The digital platform that allows you to make payments to the Public Administration and others, in a transparent and intuitive way                                                                                                                                                                                                       |
| **Service**       | A service provided by an entity                                                                                                                                                                                                                                                                                                         |
| **User**          | Person registered on the Developer Portal [**developer.io.italia.it**](https://developer.io.italia.it/), or a person authorized as an Administrative or Operational Contact for the Reserved Area [**selfcare.pagopa.it**](https://selfcare.pagopa.it/) |
