# `use` API Key

The `use` API Key is the service key you can use to [send a message](../../inviare-un-messaggio/ "mention") via API.&#x20;

## Retrieve the `use` key

### Via the Developer Portal

1. [**Log in**](https://developer.io.italia.it/) to the Developer Portal;
2. In the left column, select **“Services”**;
3. Search by Service ID or scroll through the service boxes to find the service you need
4. You will find the key at the bottom of the service box
5. View and copy the key to use it in your integration.

### Via the Reserved Area

1. [**Log in**](https://selfcare.pagopa.it/) to the Reserved Area;
2. Select the entity you want to work with from the list shown;
3. Among the active products, find the IO App and click on its box;
4. In the left column, select **“Services”**;
5. Use the search by Service ID or scroll through the list of services and select the service you need;
6. At the bottom you will find a box (_API Key_) with the service's `use` key.
7. View and copy the key to use it in your integration.

### Via API

1. Use the [manage-service-get-keys.md](../../../api-e-specifiche/api-servizi/manage-service-get-keys.md "mention") API.
2. View and copy the key to use it in your integration.
