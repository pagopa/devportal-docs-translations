# Subscription feed

{% hint style="info" %}
This feature is restricted to users of services developed for central government entities (e.g.  ministries, national-level entities, etc.).
{% endhint %}

To enable the _subscription feed,_ you must contact the system administrators at the email address [areariservata@assistenza.pagopa.it](mailto:areariservata@assistenza.pagopa.it) specifying:

<details>

<summary>If you are using the Developer Portal</summary>

- Email of the user registered to the Developer Portal

</details>

<details>

<summary>If you are using the Area Riservata</summary>

- Your Codice Fiscale as a user registered to the Area Riservata

</details>
