# Access to certificates enforceable against third parties and documents for analog of digital notification

In the details of the specific notification, the notification timeline is displayed (**Notification status**).

The events are organized as a timeline, ordered chronologically, and include the time at which each event occurred and a description of the event itself.

If a document is associated with an event, whether it is legally enforceable evidence against third parties or a document related to analog or digital notification, a link is provided that allows the user to download that document.
